# 用腦波判斷你有沒有失智——這篇論文在幹嘛？

## 一句話總結

這篇論文做了一個**又小又準的 AI 模型**，用你的**腦波（EEG）**來判斷你是健康的、得了阿茲海默症、還是額顳葉失智症，而且它還能告訴醫生「我是根據什麼做出這個判斷的」。

---

## 為什麼要做這個？

失智症（尤其是阿茲海默症和額顳葉失智症）越來越多人得，全球預估 2050 年會超過 1.3 億人。

目前要診斷失智症，通常要做 MRI、抽血、找專家看報告——**又貴、又慢、又不方便**。

腦波（EEG）檢查相對便宜、快速、不用開刀，但問題是：**光看腦波圖，人眼很難分辨「正常老化」跟「早期失智」的差別**。所以他們想讓 AI 來幫忙看。

---

## 他們怎麼做的？（流程白話版）

### Step 1：收集腦波資料

資料集是公開的，託管在 [OpenNeuro](https://openneuro.org/datasets/ds004504)（編號 `ds004504`），由希臘 AHEPA 大學醫院第二神經科收集，格式為 BIDS 標準（含原始與預處理後的 EEG）。

找了 88 個人，頭上戴 19 個電極帽，閉眼靜坐，錄腦波：
- 36 人有阿茲海默症（平均 MMSE 17.75，認知退化明顯）
- 23 人有額顳葉失智症（平均 MMSE 22.17）
- 29 人是健康的（MMSE 滿分 30）

> **MMSE（Mini-Mental State Examination，簡易智能量表）是什麼？**
>
> 一份大約 10 分鐘就能做完的紙筆測驗，醫生用來快速評估一個人的認知功能。滿分 30 分，測驗內容包括：
>
> | 項目 | 測什麼 | 範例題目 |
> |------|--------|----------|
> | 定向力（10 分） | 知不知道現在是何時何地 | 「今天幾月幾號？」「你現在在哪間醫院？」 |
> | 記憶力（6 分） | 能不能記住東西 | 聽三個詞（蘋果、桌子、硬幣），幾分鐘後問你還記不記得 |
> | 注意力與計算（5 分） | 專注力和基本運算 | 「100 減 7 等於？再減 7？再減 7？」 |
> | 語言能力（8 分） | 理解和表達 | 看一支筆，問「這是什麼？」；照指令做動作 |
> | 空間能力（1 分） | 視覺建構 | 照著畫一個交疊的五邊形 |
>
> **分數怎麼看：**
> - **27-30 分**：正常
> - **21-26 分**：輕度認知障礙
> - **10-20 分**：中度失智
> - **10 分以下**：重度失智
>
> 所以這個資料集裡，健康組平均 30 分（完全正常），阿茲海默組平均 17.75 分（中度），額顳葉失智組平均 22.17 分（輕度到中度）。

### Step 2：清理腦波（前處理）

腦波很髒，會混到眨眼、咬牙、肌肉抖動等雜訊。他們用濾波器和 ICA（獨立成分分析）把這些雜訊挑掉，只留下真正的腦部訊號。

### Step 3：把腦波變成「特徵」（特徵工程）

#### 原始資料長什麼樣？

原始 EEG 就是一堆**電壓隨時間變化的數字**。每秒取樣 500 次（500 Hz），19 個電極同時在錄，所以每秒會產生 19 × 500 = 9,500 個數值。

想像一個 Excel 表：

```
        Fp1      Fp2      F3       F4    ...（共 19 欄，每欄一個電極）
t=0.000  +12.3µV  -5.1µV   +8.7µV  +3.2µV
t=0.002  +11.8µV  -4.9µV   +9.1µV  +3.5µV
t=0.004  +10.2µV  -5.3µV   +8.4µV  +2.8µV
  ...      ...      ...      ...     ...
```

每一列是一個時間點，每一欄是一個電極的電壓值。一個人的錄製可能有好幾分鐘，就是幾十萬列的數字。

**問題是：這些原始數字直接丟給 AI，效果很差。** 需要先轉換成有意義的「特徵」。

#### PSD 和 RBP 到底是什麼？

先把兩個名詞分清楚：

| 名詞 | 全名 | 白話意思 | 在這篇論文中的角色 |
|------|------|----------|------------------|
| PSD | Power Spectral Density，功率頻譜密度 | 把一段腦波拆開來看：「每個頻率各有多少能量」 | 中間步驟，用來算頻帶能量 |
| RBP | Relative Band Power，相對頻帶功率 | 某個頻帶的能量佔全部腦波能量的比例 | 最後餵給模型的特徵 |

原始 EEG 是「時間訊號」：例如 Fp1 電極在第 0.000 秒是幾微伏、第 0.002 秒是幾微伏。這種資料看的是**電壓怎麼隨時間變化**。

PSD 則是把同一段訊號換一個角度看：不要看時間，而是看**頻率成分**。例如這 6 秒腦波裡，2 Hz 的慢波有多少能量、10 Hz 的 alpha 波有多少能量、25 Hz 的 beta 波有多少能量。

可以把它想成聲音分析：

```text
原始聲音波形：
「這段聲音的音量怎麼隨時間上下震動」

PSD：
「這段聲音裡，低音、中音、高音各有多強」
```

EEG 也是一樣：

```text
原始 EEG：
「這 6 秒內，電壓怎麼隨時間上下震動」

PSD：
「這 6 秒內，0.5 Hz、1 Hz、2 Hz、...、45 Hz 各有多少能量」
```

Welch method 不是一個新的特徵，而是**估計 PSD 的方法**。它會把一段訊號再切成更小的短片段，分別算頻譜後取平均，讓 PSD 比較穩、不容易被單一瞬間雜訊影響。

有了 PSD 之後，才開始算 RBP。RBP 不關心每一個細小頻率點，而是把頻率整理成幾個有生理意義的頻帶：

```text
Delta RBP = Delta 頻帶功率 / 0.5-45 Hz 總功率
Theta RBP = Theta 頻帶功率 / 0.5-45 Hz 總功率
Alpha RBP = Alpha 頻帶功率 / 0.5-45 Hz 總功率
...
```

例如某個 6 秒 epoch 的總功率是 100，Delta 頻帶功率是 35，那 Delta RBP 就是 0.35，意思是：**這段腦波有 35% 的能量集中在 Delta 頻帶**。

所以資料處理流程要記成：

```text
原始 EEG waveform
  -> Welch method 算 PSD
  -> 從 PSD 加總各頻帶功率
  -> 除以總功率得到 RBP
  -> RBP 才是論文模型實際吃的輸入
```

因此，PSD 是「算 RBP 之前的中間結果」；RBP 是「壓縮後、真正拿去訓練模型的特徵」。

#### 怎麼從原始腦波算出 RBP？

腦波其實是一堆不同頻率的波疊在一起（就像聲音是高音+中音+低音的混合）。他們把腦波拆成六個頻段：

| 頻段    | 頻率範圍     | 白話意思            |
| ----- | -------- | --------------- |
| Delta | 0.5-4 Hz | 深度睡眠波           |
| Theta | 4-8 Hz   | 放鬆、打瞌睡          |
| Alpha | 8-16 Hz  | 清醒但放鬆（閉眼）       |
| Zaeta | 16-24 Hz | 過渡頻段（本研究特別拆出來的） |
| Beta  | 24-30 Hz | 專注思考、認知活動       |
| Gamma | 30-45 Hz | 高階認知處理          |

#### RBP 計算流程（白話版）

```
原始腦波（時間域：電壓 vs. 時間）
  │
  │  ① 切片：把整段腦波切成每段 6 秒、相鄰片段重疊 50%
  │         （例如 0-6s, 3-9s, 6-12s, ...）
  ▼
每一個 6 秒片段（Epoch）
  │
  │  ② 傅立葉轉換 + Welch 法：把「電壓 vs. 時間」
  │     轉成「功率 vs. 頻率」（PSD，功率頻譜密度）
  │
  │     白話：算出每個頻率有多少「能量」
  │     就像把一首歌拆成「低音有多強、中音有多強、高音有多強」
  ▼
功率頻譜（每個頻率的功率值）
  │
  │  ③ 算各頻段的功率佔比（RBP）：
  │
  │                某頻段內的功率總和
  │     RBP = ─────────────────────────
  │            0.5~45 Hz 全部的功率總和
  │
  │     例：Delta RBP = Delta 功率 ÷ 總功率 = 0.35（佔 35%）
  ▼
一個電極得到 6 個 RBP 值（六個頻段各一個）
  │
  │  ④ 跨電極平均：把 19 個電極的 RBP 取平均
  ▼
一個 Epoch 得到 6 個數字 [Delta, Theta, Alpha, Zaeta, Beta, Gamma]
  │
  │  ⑤ 所有 Epoch 堆起來 → 特徵矩陣
  ▼
最終輸入：每一列是一個 Epoch，每一欄是一個頻帶的 RBP
```

**白話總結：** 原本幾十萬個電壓數字，經過這套流程，每 6 秒濃縮成 6 個數字（每個頻段佔總功率的百分比）。這 6 個數字就是餵給 AI 的「食物」。

**重點：** 他們特別把 Zaeta（16-24 Hz）獨立拆出來，這是跟其他研究不一樣的地方。傳統做法只分 5 個頻段（把 16-30 Hz 全算 Beta），但他們拆出 Zaeta 後準確率從 63% 提升到 80%，證明這個過渡頻段對判斷失智症很有用。

### Step 4：用 AI 模型分類

他們設計了一個混合模型，結合兩種深度學習架構：

- **TCN（時序卷積網路）**：擅長抓「局部的波形特徵」，像是某一小段腦波長什麼樣子
- **LSTM（長短期記憶網路）**：擅長抓「前後的時間關係」，像是腦波隨時間怎麼變化

兩個合作，先用 TCN 抓特徵，再用 LSTM 看前後文，最後判斷這個人屬於哪一類。

**模型架構（資料從上往下流）：**

```
輸入（6 個頻帶的 RBP 值）
  │
  ▼
┌─────────── TCN 區塊 ───────────┐
│ Conv1D（32 個濾波器，核心大小 7） │  ← 抓局部波形特徵
│     ↓                          │
│ Batch Normalization + ReLU     │  ← 穩定訓練
│     ↓                          │
│ Spatial Dropout（丟棄率 0.3）   │  ← 防止過擬合
│     ↓                          │
│ Conv1D（擴張卷積，擴張率 1）     │  ← 再抓一次更高階的特徵
│     ↓                          │
│ Batch Normalization + ReLU     │
│     ↓                          │
│  ＋ 殘差連接（跳接原始輸入）     │  ← 避免梯度消失
└────────────────────────────────┘
  │
  ▼
LSTM（64 個單元）                    ← 捕捉時間序列的前後關係
  │
  ▼
Dense 128 → Dropout 0.2
  ↓
Dense 192 → Dropout 0.2
  ↓
Dense 256 → Dropout 0.2
  ↓
Dense 3（Softmax 輸出）              ← 最終分成三類
```

**超參數：** 批次大小 32、Adam 優化器、學習率 0.0001，用隨機搜索調參。

整個模型只有 **13 萬個參數、佔 514 KB**，非常小，小到可以塞進手機或穿戴裝置裡。

### Step 5：用 SHAP 解釋 AI 的判斷

醫生不會信一個「黑箱」——你說他有失智症，總要告訴我為什麼吧？

**SHAP（SHapley Additive exPlanations）是什麼？**

SHAP 的核心概念來自賽局理論的 **Shapley Value（夏普利值）**。想像一個比喻：

> 一群人合作完成一個專案拿到獎金，問題是：**每個人貢獻了多少？**
>
> Shapley Value 的做法是：把每個人「加入」和「不加入」團隊時的成果差異算出來，對所有可能的加入順序取平均，就是這個人的公平貢獻值。

套到 AI 模型上：
- 「一群人」＝ 模型的各個輸入特徵（Delta、Theta、Alpha、Zaeta、Beta、Gamma）
- 「獎金」＝ 模型的預測結果
- SHAP 算的是：**每個特徵對最終預測「推了多少力」**

#### SHAP 怎麼算的？（用這篇論文的例子）

假設模型要判斷一個人是不是阿茲海默症，輸入有 6 個特徵（六個頻帶的 RBP）。我們想知道「Beta 這個特徵貢獻了多少」。

**Step 1：列出所有可能的特徵組合（不含 Beta）**

6 個特徵中拿掉 Beta，剩下 5 個可以組成各種子集：

```
{}                          ← 什麼特徵都不給
{Delta}                     ← 只給 Delta
{Delta, Theta}              ← 給 Delta + Theta
{Delta, Theta, Alpha}       ← 給三個
...
{Delta, Theta, Alpha, Zaeta, Gamma}  ← 給除了 Beta 以外全部
```

共 2⁵ = 32 種子集。

**Step 2：每個子集，比較「加入 Beta 前」和「加入 Beta 後」的預測差異**

```
子集 S                        不含 Beta 的預測    含 Beta 的預測     差異（邊際貢獻）
─────────────────────────────────────────────────────────────────────────
{}                            0.33（亂猜）       0.45              +0.12
{Delta}                       0.35              0.52              +0.17
{Delta, Theta}                0.40              0.61              +0.21
{Delta, Theta, Alpha, Zaeta}  0.55              0.78              +0.23
...全部 5 個                   0.62              0.85              +0.23
```

（以上數字是示意，非論文實際數值）

**Step 3：加權平均所有邊際貢獻**

不是直接算平均，而是用 Shapley 公式加權：

```
              1
φ_Beta = ─── × Σ  [子集大小的權重] × [加入 Beta 後的預測 - 加入前的預測]
             6!

權重 = |S|! × (6-|S|-1)! / 6!

其中 |S| 是子集的大小，6 是總特徵數
```

白話：**小子集和大子集的權重比較高，中等大小的子集權重比較低**。直覺是：在「幾乎沒有隊友」和「幾乎全員到齊」時加入的貢獻，比較能反映一個人的真實價值。

**Step 4：得到 SHAP 值**

最終算出來的加權平均就是 Beta 的 SHAP 值，例如 +0.19。

**所有特徵的 SHAP 值加起來 = 模型預測值 - 基準值（平均預測）**，這是 SHAP 的核心性質。

```
舉例（示意）：
基準值（平均預測「是阿茲海默」的機率）= 0.41

某個病人的預測 = 0.85

SHAP 拆解：
  Delta:  +0.02  ■
  Theta:  +0.06  ■■■
  Alpha:  -0.03  ■■（往反方向推）
  Zaeta:  +0.12  ■■■■■■
  Beta:   +0.19  ■■■■■■■■■■
  Gamma:  +0.08  ■■■■
  ─────────────────
  總和:   +0.44  →  0.41 + 0.44 = 0.85 ✓
```

**白話總結：** SHAP 就是把模型的預測拆成每個特徵的「推力」。正值代表把預測往該類別推，負值代表往反方向推。所有推力加起來剛好等於預測結果和基準值的差距。

> **為什麼不直接看特徵的權重就好？**
>
> 因為深度學習模型不像線性迴歸有明確的權重。神經網路的權重散布在好幾層，彼此交互作用，直接看沒有意義。SHAP 是從「輸入和輸出的關係」下手，不管模型內部多複雜，都能算出每個特徵的貢獻。

舉例來說，模型判斷某人是阿茲海默症，SHAP 可以告訴你：
- Beta 頻帶把預測往「阿茲海默」推了 +0.19（最大推力）
- Delta 頻帶幾乎沒影響（推力接近 0）

**這篇論文用了三種 SHAP 圖表：**

| 圖表            | 看什麼                               |
| ------------- | --------------------------------- |
| 全域特徵重要性圖      | 整體來看，哪個頻帶最重要（平均 SHAP 值排序）         |
| 摘要圖（Beeswarm） | 每個樣本的 SHAP 值分布，可以看到「特徵值高/低」如何影響預測 |
| 熱力圖           | 跨樣本的 SHAP 值變化趨勢，一眼看出哪些頻帶在哪些樣本特別突出 |

**這篇論文的 SHAP 結果：**
- 判斷阿茲海默症時，模型最看重 **Beta（+0.19）和 Zaeta（+0.12）** 頻段
- 判斷額顳葉失智症時，也是 **Beta（+0.26）和 Zaeta（+0.2）** 最重要
- 判斷健康時，**Zaeta（+0.1）** 最關鍵，其次是 Beta 和 Theta
- 這跟神經科學的知識吻合：Beta 跟認知功能有關，失智症患者的 Beta 波確實會異常；Zaeta 作為過渡頻段，能捕捉到傳統分法忽略的微妙變化

| 任務 | 準確率 |
|------|--------|
| 阿茲海默症 vs. 健康 | **99.7%** |

二元分類（有病 vs. 沒病）幾乎完美。三類一起分比較難，主要是阿茲海默症跟健康的腦波有時候長得比較像。

## 這篇論文的價值在哪？

1. **便宜好用**：腦波儀比 MRI 便宜太多，而且這個模型小到能跑在邊緣裝置上
2. **可以解釋**：醫生能看到 AI 是根據哪些腦波特徵做判斷的，增加信任感
3. **改良的特徵工程**：把 Zaeta 頻段獨立出來，比傳統做法（63%）大幅提升到 80%
4. **實用潛力**：未來有機會做成穿戴式裝置，在社區篩檢或居家監測中使用

## 限制

- 樣本只有 88 人，偏少
- 三類分類的準確率（80%）在臨床上還不夠好
- 只測了兩種失智症，還有其他類型沒涵蓋
- 沒有在不同醫院、不同設備上驗證過
