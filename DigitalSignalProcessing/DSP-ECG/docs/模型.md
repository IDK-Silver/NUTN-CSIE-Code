# 模型

## TCN-LSTM Classifier

`src/ecg/models/tcn_lstm.py` 定義 `TcnLstmClassifier`。

模型流程：

```text
TCN residual blocks -> LSTM -> dense classifier
```

輸入 tensor 形狀固定使用：

```text
[batch, seq_len, input_dim]
```

其中：

- `seq_len` 不固定，可以是 frequency bands、time steps，或其他明確定義的序列長度。
- `input_dim` 不固定，由呼叫端建立模型時明確指定。
- `num_classes` 不固定，由呼叫端建立模型時明確指定。

模型不提供關鍵參數 default。建立模型時必須明確指定 `input_dim`、`num_classes`、TCN channels/kernel/dilations/dropout、LSTM hidden size、dense hidden layers/dropout。

`TcnLstmClassifier` 不知道 `ds004504`、`ds004504_rbp_paper`、6 bands、3 classes，也不負責 H5 讀取或 feature reshape。這些資料語意應該放在 Dataset 或 transform 層。
