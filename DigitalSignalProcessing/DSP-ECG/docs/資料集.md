# 資料集

## H5 Reader

`H5FeatureLabelReader` 是共用 H5 reader，只負責從同一個 H5 檔讀取兩個 row-aligned datasets。

建立時必須明確指定：

- `h5_path`
- `feature_key`
- `label_key`

它不理解 `ds004504`、frequency band、class label，也不做 feature reshape。

## Ds004504RbpPaperDataset

`Ds004504RbpPaperDataset` 只支援 `ds004504_rbp_paper` 這個 processed dataset。它會讀取 command 產生的 H5 與 manifest：

```python
from pathlib import Path

from ecg.datasets import Ds004504RbpPaperDataset

dataset = Ds004504RbpPaperDataset(
    h5_path=Path("data/processed_raw_dataset/ds004504_rbp_paper.h5"),
    manifest_path=Path("data/processed_raw_dataset/ds004504_rbp_paper_manifest.json"),
)
```

這個 Dataset 固定讀：

- feature: `X_rbp_mean`
- label: `y`

回傳：

```text
x shape: [n_bands, 1]
y shape: []
```

原因是論文模型使用跨 channel 平均後的 RBP band sequence，對應 H5 裡已經預先寫好的 `X_rbp_mean`。`X_rbp_channel` 保留在 H5 中供後續分析或新方法使用，但不屬於 `ds004504_rbp_paper` Dataset 的模型輸入。

如果之後要使用 per-channel feature，例如 `[n_channels, n_bands]` 或 flatten 成 `[n_channels * n_bands]`，應另外定義新的 dataset-specific Dataset 或 transform，避免讓 paper Dataset 同時承擔不同方法的 feature layout。
