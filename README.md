# 國立臺南大學資訊工程學系課程程式碼

本專案收錄國立臺南大學資訊工程學系各課程的作業、專案與實驗程式碼。

## 科目總覽

| 科目 | 資料夾 | 內容 |
|------|--------|------|
| 演算法 | [Algorithm](Algorithm/) | 質數演算法、電動車馬達最佳化、樹結構 |
| 雲端運算 | [CloudComputing](CloudComputing/) | 雲端服務實作 |
| 電腦圖學 | [ComputerGraphics](ComputerGraphics/) | OpenGL 作業與期末專案 |
| 計算機程式設計 | [ComputerProgramming](ComputerProgramming/) | 程式設計基礎練習 |
| 資料探勘 | [DataMining](DataMining/) | 期中專案、期末專案（糖尿病分類器） |
| 資料結構 | [DataStructure](DataStructure/) | 計算機、迷宮、二元搜尋樹、圖論 |
| 數位影像處理 | [Digital Image Processing](<Digital Image Processing/>) | 手寫處理、影像修復、期中期末專案 |
| 數位系統 | [DigitalSystem](DigitalSystem/) | 數位電路作業 |
| 物件導向程式設計 | [ObjectOrientedProgramming](ObjectOrientedProgramming/) | OOP 作業與練習 |
| 機率論 | [Probability](Probability/) | 統計分析與視覺化 |
| 訊號與系統 | [SignalsAndSystems](SignalsAndSystems/) | 訊號處理作業與期末專案 |
| 軟性計算 | [SoftComputing](SoftComputing/) | DPSO 演算法實作 |
| 軟體工程 | [SoftwareEngineering](SoftwareEngineering/) | 軟體開發專案與作業 |
| 系統軟體 | [SystemSoftware](SystemSoftware/) | 絕對載入器實作 |
| 無線通訊網路 | [WirelessCommunicationNetworks](WirelessCommunicationNetworks/) | BLE 廣播、Tello 無人機控制 |

## 快速開始

取得原始碼：

```bash
git clone https://github.com/IDK-Silver/NUTN-CSIE-Code.git
cd NUTN-CSIE-Code
```

各科目的詳細說明請參考各資料夾內的 README。
